## [1.0.0] - 2021-10-25
### Added
- Support for all block explorer commands:
  - getAccount,
  - getAccounts,
  - getBlock,
  - getBlockIds,
  - getBroadcast,
  - getMe,
  - getMessage,
  - getMessageIds,
  - getTransaction.
- Support for basic transactions:
  - broadcast,
  - sendMany,
  - sendOne.
- ADS utils
- Readme

- Changelog
- Contributing
